## 返回值|运算类 

### JavaScript中有 6 种值为“假”
```
1.false (布尔型)
2.null (用于定义空的或者不存在的引用)
3.undefined (未定义值)
4.0 (数值型)
5."'' (空字符串) (字符型)
6.NaN
//
// -1 布尔值为true  ' ' 有空格字符串 布尔值为true
```
### ~运算符  按位取反
所有正整数的按位取反是其本身+1的负数，所有负整数的按位取反是其本身+1的绝对值，零的按位取反是 -1；其他非数字或不能转化为数字类型的返回值，统一当做0处理,返回-1
